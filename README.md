# How to see this?

Use the golang present tool 
```
# setup GOPATH before running this: https://golang.org/doc/code.html#GOPATH
# Install presenter: https://github.com/audreylim/go-presenter#install
cd /path/to/this/slide
$GOPATH/bin/present
```
